克里米亚之旅-塞瓦斯托波尔

发表于 2016-03-08   |   分类于 游   |     |   次阅读

莫斯科->辛菲罗波尔

辛菲罗波尔是克里米亚自治共和国的首府，好像整个半岛只有在辛菲罗波尔有民用机场。从莫斯科到辛菲罗波尔大概1400公里，飞机两个半小时。我们这次坐的是俄罗斯当地的VIM航空公司，从莫斯科南边的多莫杰多沃机场（Domodedovo，Домодедово），该机场是莫斯科最大的机场，距离市区比较远，开车需要一个小时左右。VIM航空感觉有点类似于廉价航空，下图的飞机餐大家可以感受一下。

辛菲罗波尔->塞瓦斯托波尔

辛菲罗波尔机场挺新，不过很小，没有行李托运的话很快就可以出来。

机场大厅里面有出租车司机搭讪，不过最好不要在这里坐出租车，宰人太厉害。我们稍微问了一下价格，从辛菲罗波尔到塞瓦斯托波尔报价要5000卢布，后来又降到3000卢布。大厅里面还有一个租车的地方，看了一下好像很划得来，普通的车租一天只要两三千卢布，有自驾游兴趣的很值得一试。从机场大厅出来对面就可以看到卖长途汽车票的地方，到各个城市的票价如下，基本都在150卢布以内。

这里也有很多出租车及黑车司机拉客，一般到塞瓦斯托波尔价格在2000卢布以内，我们还了一下价，最终1500卢布，如果人多这个价格还是很划得来的，毕竟租车要快很多。
值得一提的是，从辛菲罗波尔到雅尔塔有世界上最长的城际无轨电车线路（117公里），1959年投入运行，至今仍在服役，有时间的话尝试一下也很不错。

从辛菲罗波尔去塞瓦斯托波尔的路况并不是很好，很多时候都是两车道的乡间公路，有些时候路面还坑坑洼洼，感觉有比较长的时间没有修理了。不过路两旁的风景很好，现在这个时候虽然很多树还是光秃秃的，但有些已经开始吐芽，有些草地也开始变绿，白色的梨树花已经盛开，整齐的葡萄园一望无际，再加上蓝天白云，让我们这些刚从整天阴郁的冰雪莫斯科来的人实在心情舒畅。
不过一路还是可以看到政局的不稳给当地带来的影响，很多工厂都停工了，路旁废弃的房子比比皆是，公交车基本上都比我的年龄大。我一直很好奇，克里米亚从乌克兰回归俄罗斯之后，民众手中的钱怎么处理？银行、电信、水电等基础设施转换？税务、司法、财政、工商怎么衔接？问了一下司机他们原来的乌克兰钱怎么处理，司机摆摆手没有说什么，好像是说全没了？不过我觉得不太可能。路两边还有很多没有建成的小房子，有些建了一半就开始住人，起初我以为是因为乌克兰危机，房子的主人把修到一般的房子废弃了，不过后来听司机说，这是鞑靼人的房子。当年克里米亚的鞑靼人被斯大林流放到中亚地区，但在克里米亚建立过汗国的鞑靼人对克里米亚也是旧情难断，最近几年开始不断地有鞑靼人回归克里米亚，并搭建自己的房子，有些用地是被政府批准的，有些是私自占地。

塞瓦斯托波尔：沉船纪念碑

我们的宾馆是在booking上定好的，现在是旅游淡季，价格很便宜，标间不到3000卢布一晚，位置很好，就在塞瓦斯托波尔中心，塞瓦斯托波尔海港旁边。离塞瓦斯托波尔的标志性建筑-沉船纪念碑很近。沉船纪念碑是为纪念克里米亚战争(1853-1856)中俄军为阻挡英、法、土联军进入塞瓦斯托波尔港弄沉的15艘舰船而建。“沉船纪念碑”在经历塞瓦斯托波尔保卫战中德军地毯式轰炸后居然奇迹般地保存了下来。

这个海港也是俄罗斯的黑海舰队所在地，据说可以坐船绕着军港看一圈，不过当天我们到的时候已是傍晚，没有看到邮轮，只看到两艘军舰静静地停在海港里面。

海港旁边有个小公园，里面有介绍黑海舰队的宣传板，写着“我们的骄傲”。

过马路就是二战纪念碑。

塞瓦斯托波尔：赫尔松涅斯古希腊遗址（Херсонес）

赫尔松涅斯在古希腊语中意为“半岛”，该遗址古城位于克里米亚半岛西南侧，两千多年前由希腊人所建，当时是克里米亚半岛重要的政治经济文化中心。整个遗址保存了不少古代建筑的残垣断壁。


遗址离市中心不是很远，可以坐公交车到。克里米亚的公交都很便宜，塞瓦斯托波尔的基本都是12卢布，其他城市的市内公交也都是十来卢布起价。

从下面的柱子可以看出是古希腊的风格：

古希腊遗址旁边有一个教堂，圣弗拉基米尔大公大教堂，这个教堂的规模虽然与莫斯科的那些大教堂相比，但在克里米亚半岛来说基本是最大的了。比较特别是这里埋葬着1853年-1856年克里米亚战争的三位海军将领。
该遗址就在海边，风景很是优美，海水特别清澈。

塞瓦斯托波尔：巴拉克拉瓦港(Балаклава)

巴拉克拉瓦港位于塞瓦斯托波尔的东南角，离城区不是很近，打车大概四五百卢布。但我认为它绝对是塞瓦斯托波尔最值得去的地方。
克里米亚战争中巴拉克拉瓦就是著名的巴拉克拉瓦战役所在地。在第二次世界大战中，巴拉克拉瓦是苏德线的最南端。巴拉克拉瓦海湾，间于山之间，风景如诗如画，海湾两侧有船坞，秘密苏联潜艇基地，古防御城堡，100年前，著名作家库普林的小说也诞生在这里。此地区是苏联秘密住宅区之一，潜艇基地工作人员几乎是此地区的全部人口。1991年苏联解体后，直到1993年潜艇迁出，此地区才得以对外开放。

虽然海港两旁住了不少人，但是海水非常清澈，里面可以看到很多小鱼，很多当地人在岸边或者船上钓鱼，我觉得这完全不像是钓鱼，因为鱼实在太多，感觉只要随便勾就可以。

到了巴拉克拉瓦，一定要爬到旁边的山顶上，可以俯瞰整个海港：

最为重要的是，在山顶可以看到山那边的黑海，陡峭的悬崖、碧蓝的海水，没有什么比这个更让人心胸舒畅的了。

塞瓦斯托波尔：Fiolent海角

Fiolent海角去的人比较少，基本上还没有看到有中文的介绍。我们也是在巴拉克拉瓦看到卖画的摊位上有一幅很漂亮的画，然后问摊主这是那里，摊主告诉我们是Fiolent，于是我们决定前往。去Fiolent有公交车，不过需要先从巴拉克拉瓦坐车到总车站再转车。

到了Fiolent海角之后如果要到沙滩上需要爬800个台阶，虽然很辛苦，但是很值得，在这里你可以静静地享受海浪和涛声，有很多各色的小石子，我捡到一块貌似玛瑙的小石头，朋友捡到一块疑似化石的骨头。。。

Fiolent海角旁边有个小岛，岛上竖立着一个十字架，如果是傍晚的话，在夕阳底下特别漂亮：

Fiolent海角还有一座教堂，在这个教堂旁边我居然看到有竹子，这是我在俄罗斯其他地方没有见到过的。
