克里米亚之旅-雅尔塔

发表于 2016-03-09   |   分类于 游   |     |   次阅读

雅尔塔对于任何一个上过初中历史的人来说，都是极为熟悉的，但我没有想到有一天还能身临其境。

雅尔塔位于克里米亚半岛南岸,世界闻名的历史古城，建于12世纪。人口约有141781人，面积283km²。也是黑海沿岸知名的疗养区。雅尔塔从19世纪开始发展旅游业。在20世纪，雅尔塔是苏联最重要的疗养区。1945年的雅尔塔会议就是在这里举行。

雅尔塔作为旅游胜地已经一百多年了，必然有其独特的地方，山、海、人文景观都应有尽有。因为乌克兰危机，目前雅尔塔的游客减少了差不多一半，所以这个时候去雅尔塔很是划算。我们住的海景房，离海边不到十米，坐在阳台上就可以看到大海，每天晚上听着海浪声入睡，标准间才四千多卢布一晚。
第一天早上在醒来刚好赶上日出，感觉在阳台上拍了几张日出照：

雅尔塔：里瓦几亚宫

在雅尔塔本来准备第一个景点去燕子堡，google地图查的公交路线，结果坐到一半换乘时，被当地热心的俄罗斯大妈告知，去燕子堡的路现在不通，坐公交没法去了。她推荐我们去附近的里瓦几亚宫，跟着她走就可以。热心的大妈在路上表示她很不好意思，外国人到了他们这里，居然不能乘公共交通去景点。感觉克里米亚的居民和莫斯科的俄罗斯人还是有很大不同的，克里米亚人都特别的热心，第二天当我们在路上走有点犹豫往哪里走时，旁边散步的一对大爷大妈主动问我们要去哪里，然后两人商量了一会告诉我们怎么去。也许克里米亚人经历了太多，变得更加的包容。

里瓦几亚宫在半山腰，可以眺望半个雅尔塔，周边是一个公园。在克里米亚发现一个很有趣的地方，这里的花草树木和国内的特别像，松柏很多，也许气候差不多吧。我们去的时候，不知道为什么买门票的地方没人，有可能是放假了，因此我们也没有进到里面去，在旁边逛一下也挺不错。从里瓦几亚宫还可以沿着小路一直下到海边。周边还有一些私人住宅，估计都是富商巨贾的。

这个地方主要是来感受一下历史氛围，虽然周边风景也很漂亮，但雅尔塔有很多这种漂亮的地方，所以也就不显得出众了。

雅尔塔：燕子堡

雅尔塔必去的景点那肯定是燕子堡了。

燕子堡是一个装饰性的城堡，位于加斯普拉，克里米亚雅尔塔和阿卢普卡之间有一个温泉小镇。它由俄罗斯建筑师列昂尼德·舍伍德（他的父亲弗拉基米尔·舍伍德是莫斯科红场上历史博物馆的建筑设计师），为波罗的海的德国石油百万富翁 Baron vonSteingel设计的一个新哥特式建筑，1911年-1912年间建在40米（130英尺）高的海边悬崖峭壁之上。该建筑所处的位置，使人一看就明白“燕子堡”这个称呼的含义。如今已成为雅尔塔具有代表性的景观。
第一次世界大战初期，莫斯科商人舍拉普京买下了燕子堡，他开设了城堡餐厅。但不久商人去世，餐厅关闭。20世纪30年代，这里成为当地度假屋的阅览室，后因发生紧急情况而关闭。20世纪70年代重建之后，这里成为一间餐厅，燕子堡开始被认为是克里米亚的象征。

雅尔塔：阿卢普卡沃龙佐夫宫

从燕子堡上来，本来是想去坐缆车到艾彼得山顶的，不过当我们到了坐缆车的地方才发现冬季缆车不工作，只能远远看一眼艾彼得山。

还好离坐缆车不远的地方就是阿卢普卡沃龙佐夫宫（Воронцовский дворец），于是步行前往沃龙佐夫宫。

阿卢普卡沃龙佐夫宫由沃龙佐夫，米哈伊尔·谢苗诺维奇始建于19世纪，摩尔复兴建筑风格。建筑用的辉绿岩在附近开采,现在该宫殿已经成为博物馆。沃龙佐夫宫四周是个大公园——它是景观艺术的丰碑。从1824年12月至1851年4月，一位有才华的德国园艺家植物学家 卡尔.安东诺维奇.科巴赫——克里米亚南部海岸首席园艺学家，在阿卢普卡创造了沃龙佐夫公园。 沃龙佐夫宫的修建，运用了新的(相对于古典主义)建筑设计和施工原则。一个重要的建筑特点是宫殿的选址与山区地形区别开，使得宫殿非常有机地融入周围的风景，展现自己原有的艺术性和具表现力的外观。 宫殿建筑具英式建筑风格，且有不同时代的元素。开始于早期风格，结束于十六世纪风格。时代元素排列开始于西大门，越远离西大门，时代元素风格越距离现代近。 英式风格有机地结合了摩尔复兴建筑风格。例如：哥特式的烟囱让人联想到清真寺的尖塔。南入口装饰有东方的异彩，马蹄拱，双层拱，壁龛有石膏雕花，其中交织绘有都铎花与莲花图案，重复6次完成了阿拉伯题词楣：“除了真主没有胜利者”。

一天的时间在雅尔塔游玩这些地方就已经足够了，时间充裕的话随便在海边走走，发发呆，在公园里面坐坐，或者与好友在海边的酒吧里面品尝一下当地马桑德拉酒庄的葡萄酒，也是一种享受。遗憾的是这次没能去参观马桑德拉酒庄博物馆，不过这也给下次再来留下念想。来到雅尔塔这种地方，总是有种在当地定居养老的冲动。

从雅尔塔回辛菲罗波尔，时间充裕的话可以体验一下世界上最长的无轨电车，或者到汽车站坐长途汽车，更方便的是直接打车去机场，两千卢布左右，现在卢布贬值厉害，用起来不怎么心疼了。
